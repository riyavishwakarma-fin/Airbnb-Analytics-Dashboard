# Airbnb-Analytics-Dashboard
Power BI dashboard analyzing Airbnb pricing, location trends, and listing behavior.📊
This project explores Airbnb listing data using Power BI to uncover insights related to pricing, location, and listing distribution.
#🔍 Key Insights
Manhattan has the highest average listing price
Entire homes dominate the listings (~52%)
Brooklyn and Manhattan lead in listing activity
Pricing varies significantly across room types
#🛠️ Tools Used
Power BI
DAX
#🚀 Project Goal
To transform raw Airbnb data into meaningful insights through data visualization and storytelling.
